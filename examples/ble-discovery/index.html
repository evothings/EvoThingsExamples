This app automatically scans for BLE devices. Note that you must run this app in the EvoThings client app, or build a Cordova app with the plugin com.evothings.ble. Device with support for Bluetooth 4.0 and Android 4.3 or higher required.

How to use: The app should detect running BLE devices. There are known problems with BLE stop working on multiple reloads. Restart the EvoThings client app in this case.
